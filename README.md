PMGSY – Physical and Financial Performance Analysis
📌 Project Overview

This project analyzes the Pradhan Mantri Gram Sadak Yojana (PMGSY) scheme using Power BI to evaluate both physical progress and financial performance of rural road infrastructure projects across India. The dashboard transforms raw government data into meaningful visual insights for better planning and decision-making.

🎯 Objectives
Analyze sanctioned vs completed road works.
Evaluate expenditure trends across states and schemes.
Compare state-wise infrastructure performance.
Measure road completion efficiency.
Study bridge construction progress.
Build interactive dashboards for decision support.
📂 Data Source
Government of India Open Data Portal (data.gov.in)
PMGSY official data related to:
Road length sanctioned/completed
Expenditure incurred
Cost per KM
Bridges sanctioned/completed
State-wise and scheme-wise records

🔍 Key Insights
PMGSY achieved significant road development progress.
Performance varies across states.
Higher expenditure does not always guarantee better completion rates.
Some states show strong implementation efficiency.
PMGSY-II is one of the most resource-intensive schemes.
Bihar leads in bridge sanctions.
